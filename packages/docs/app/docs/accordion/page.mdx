# Accordion

The Accordion component uses Radix UI's Accordion primitive as its base, styled with EH design tokens. It supports both single-expand and multi-expand modes.

## Design tokens used

- **Container**: `rounded-lg`, `default-ring-bezel`, gap via `--eh-spacing-*`
- **Trigger**: Styled to look like an EH Button (no nested Button component)
- **Animation**: Custom keyframes in `tokens.css` for expand/collapse
- **Base**: `@radix-ui/react-accordion`
