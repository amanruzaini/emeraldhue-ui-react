# Button

The Button component is the primary interactive element in EH. It supports 4 variants, 4 sizes, and optional leading/trailing icons.

> **Note**: The Button uses a code-first API with `children` for label text. The Figma Code Connect file (`Button.figma.tsx`) handles translation between Figma properties and these React props.

## Design tokens used

- **Colours**: `bg-brand`, `bg-brand-inverted`, `text-all-white`, `text-default`, `border-brand`
- **Border**: Asymmetric bezel — `border-t-xs border-l-xs border-r-xs border-b-sm`
- **Ring**: `default-ring-bezel` for the EH action ring + depth shadow
- **Spacing**: `px-[var(--eh-spacing-*)]` for all internal padding
- **Icon sizing**: `[--icon-size:var(--eh-icon-*)]` CSS custom property
