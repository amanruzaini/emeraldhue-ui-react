# Button

The Button component is the primary interactive element in EH. It supports 4 variants, 4 sizes, and optional leading/trailing icons.

> The Button uses a code-first API. The Figma Code Connect file (`Button.figma.tsx`) translates between Figma's property naming and these React props.

## Design tokens used

- **Colours**: `bg-brand`, `bg-error`, `text-all-white`, `text-strong`, `border-brand-strong`
- **Border**: Asymmetric bezel — `border-t-xs border-x-xs border-b-{sm|md|lg}`
- **Ring**: `default-ring-bezel` for the EH action ring + depth shadow
- **Spacing**: Variant/size compound variants control padding
- **Icon sizing**: `size-icon-sm` (xs/sm) and `size-icon-md` (md/lg)
