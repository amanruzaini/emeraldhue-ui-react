# Emerald HUE

**Emerald HUE (EH)** is a React component library built for Petronas' industrial digital products — from oil rigs to control centres.

## What makes EH different

- **6 theme modes** — Light, Dark, Light High-Contrast, Dark High-Contrast, Light Colour-Blind, Dark Colour-Blind
- **5 text resize levels** — 75%, 100%, 125%, 150%, 200%
- **WCAG 2.1 AA** compliance across all themes
- **Industrial-grade** — glove-friendly touch targets, high-contrast for outdoor use
- **Design token driven** — no hardcoded values, every visual decision flows from tokens
